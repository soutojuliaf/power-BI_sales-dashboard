# Financial & Sales Performance Dashboard

## Project Overview
This Power BI dashboard was designed to provide a comprehensive view of sales performance and financial health. As a professional with a background in **Audit and Administration**, I focused on creating a tool that not only tracks revenue but also monitors profit margins and identifies operational inconsistencies.

## Key Features
*   **Dynamic KPIs:** Real-time tracking of Gross Sales, Net Profit, and Margin %.
*   **Trend Analysis:** Month-over-Month (MoM) performance comparison.
*   **Audit Perspective:** Identification of transactions with unusual margins for risk assessment.
*   **DAX Measures:** Implementation of custom calculated measures for precise financial reporting.
